    <h1>Contribute to OrionHealth</h1>
    <p>We welcome contributions from the community! Whether it's code, documentation, or design, your help is appreciated.</p>

    <h2>How to Contribute</h2>
    <ol>
        <li>Fork the repository on GitHub.</li>
        <li>Clone your fork locally.</li>
        <li>Create a new branch for your feature or fix (e.g., <code>feat/new-chart</code>).</li>
        <li>Commit your changes and push to your fork.</li>
        <li>Submit a Pull Request to the main repository.</li>
    </ol>

    <h2>Development Setup</h2>
    <p>You will need the Flutter SDK installed. Run <code>flutter pub get</code> to install dependencies and <code>flutter run</code> to start the app.</p>

    <h2>License</h2>
