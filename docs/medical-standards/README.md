# Medical Standards Reference Data

This directory contains curated medical reference data for clinical interoperability.
All data is public, verifiable, and linked to official sources and Wikipedia for context.

## Available Standards

- [ICD-10](./icd10/) - Clinical diagnoses and classifications
- [LOINC](./loinc/) - Laboratory and clinical observations
- [RxNorm](./rxnorm/) - Clinical drug terminology
- [SNOMED CT](./snomed/) - Comprehensive clinical terminology
- [Guidelines](./guidelines/) - Clinical practice guidelines

## Sources and Verifiability

Data is sourced from:
- WHO (ICD-10)
- Regenstrief Institute (LOINC)
- NLM (RxNorm)
- SNOMED International (SNOMED CT)

## Principles

1. **FAIR**: Findable, Accessible, Interoperable, Reusable.
2. **Public**: No hidden data.
3. **Verifiable**: Direct links to official sources.
