# isar_agent_memory

Graph-based agent memory for local-first RAG on Flutter.

Built on IsarDB. Provides:
- MemoryGraph — hybrid vector + knowledge graph
- Hierarchical RAG (HiRAG) — multi-layer summarization
- Hybrid Search — semantic vector + BM25 keyword via RRF
- Medical context aware
